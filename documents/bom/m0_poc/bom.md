## 共振C組み合わせ候補

3D11-722J（L ≈ 7.2mH）の場合:

| 組み合わせ | 合計容量 | 共振周波数（理論） | 備考 |
|------------|----------|-------------------|------|
| 8200pF単体 | 8200pF | 20.6kHz | |
| 8200+560pF | 8760pF | 20.0kHz | ★ 理論最適 |
| 8200+1500pF | 9700pF | 19.0kHz | |
| 6800pF単体 | 6800pF | 22.9kHz | |
| 6800+560pF | 7360pF | 22.0kHz | |
| 6800+1500pF | 8300pF | 20.5kHz | |

※ 実測で受信振幅が最大になる組み合わせを採用
